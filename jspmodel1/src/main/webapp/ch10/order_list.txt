
액션 태그 <jsp:useBean> 
자바빈 이라고도 불린다. 

용어 정리: 
자바빈을 사용할 때  JSP 에서는 액션태그 <jsp:useBean> 을 사용한다.

  
개념과 사용방법 

회원가입 --> 데이터 베이스 공간에 저장(컬럼에 , id, 이메일) ...
데이터를 String --> "아이디홍길동비번1234이메일aaa@naver.com" 

------------------
구분하기 힘들다 
------------------

반대로 가지고 올때도 String으로 가지고 온다고 하면 10명일 경우
"아이디홍길동비번1234이메일aaa@naver.com"
"아이디홍길동비번1234이메일aaa@naver.com"
... 
String --> 배열에 담아서 온다고 하면 불가 !! 
배열을 고정길이를 먼전 선언해야 합니다. 하지만 데이터 베이스에 몇 명이 있는지 알 수 없다. 
그럼 가변배열을 사용해야 됩니다. (ArrayList<String> ) 

ArrayList<String> --> ("아이디,홍길동비번1234이메일aaa@naver.com",
								"아이디,홍길동비번1234이메일aaa@naver.com") 
추출할때 구분해서 처리하기가 힘들다. (굳이 한다면 , 약속해서 처리) --> 불가능  

그래서 우리는 데이터 베이스에 컬럼과 맵핑되는 변수(멤버변수)만 있는 클래스를 
만들기로 합니다. 그리고 그 변수에 데이터 베이스에 있는 값을 하나씩 넣을 수 있는 
기술을 사용하기로 합니다. 그리고 그 클래스를 가변길이인 ArrayList에 저장 하기로 합니다. 

ArrayList<User> <-- 이것을 자바 진영에서는 자바빈즈라고 부릅니다. 

결론 : 여러개의 데이터를 하나의 클래스에 담고 그 클래스를 (List, Map, Set)자료구조에 
저장해서 사용할 대 자바빈즈라는 용어를 자주 사용한다. 

::: 처음 사용하는 사람들은 당연히 이해하기 힘들다

자바빈즈라는 기술을 사용한다.

(JSP 프로그래밍 이라고 부르고 SPRING은 프레임워크 라고 합니다)

 위해서 사용한다.
